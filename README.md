# GraphDSL

Graph DSL for basic graph algorithms (such as **SSSP, BC, PR, and TC**) as Phase1 (static).
It contains the following steps :-

1. Generating the grammar using Lex and YACC 
2. Generating IR. 
3. Generate code from IR. 
4. Optomize the IR.
